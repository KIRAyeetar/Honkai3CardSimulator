# 崩坏3 抽卡概率模拟器
#### 啊，想起抽卡都来气~~ 让我们试试这个概率有多坑爹..

毕业指包括圣痕和武器毕业，计算了10次保底、武器60次保底以及许愿池功能。<br/>
国服机制有点特殊，分为公示概率和非保底概率（如果传闻没错）。即**十连保底+非保底概率=公示概率**。<br/>
所以内有一份pro_geter.py用于暴力计算非保底概率。
